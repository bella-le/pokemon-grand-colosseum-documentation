    Format Template

    Each trainer entry should follow this format:

    ### Trainer Name (ID)
Location: [Area]
Reward: [Money/Items if applicable]

#### Pokemon 1
- Level: XX
- Nature: XXX
- Ability: XXX
- Held Item: XXX
- EVs: XXX HP / XXX Atk / XXX Def / XXX SpA / XXX SpD / XXX Spe
- Moves:
  - Move 1
  - Move 2
  - Move 3
  - Move 4

[Repeat for each Pokemon in team]
